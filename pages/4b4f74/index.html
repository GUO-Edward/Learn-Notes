# 三次握手

为什么不是两次或四次？

如果只有两次 只会有服务器意识到建立连接 而服务端不会后续发送数据

没有四次的必要 浪费资源

# 四次挥手

为什么不能是三次？

(会产生资源的浪费) 第二次和第三次中间会有时间间隔 如果合二为一 时间超过30s会一直重新发送请求
